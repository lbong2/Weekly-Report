# 주간보고 시스템 - 화면 설계서

## 개요

SM 프로젝트 주간보고서 웹 시스템의 화면 설계 문서입니다.

---

## 화면 목록

| 구분 | 화면명 | 경로 | 권한 |
|------|--------|------|------|
| 인증 | 로그인 | /login | 비로그인 |
| 메인 | 주간보고서 목록 | / | 로그인 |
| 보고서 | 주간보고서 상세 | /reports/[id] | 로그인 |
| 관리자 | 사용자 관리 | /admin/users | ADMIN |
| 관리자 | 팀 관리 | /admin/teams | ADMIN |
| 관리자 | 모듈 관리 | /admin/chains | ADMIN |
| 관리자 | 출결유형 관리 | /admin/attendance-types | ADMIN |

---

## 1. 로그인

### 화면 구성

```
┌─────────────────────────────────────────────────────────────┐
│                         로그인                               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    [회사 로고]                               │
│                  주간보고 시스템                              │
│                                                             │
│                 ┌─────────────────────┐                     │
│     이메일      │                     │                     │
│                 └─────────────────────┘                     │
│                 ┌─────────────────────┐                     │
│     비밀번호    │                     │                     │
│                 └─────────────────────┘                     │
│                                                             │
│                 ┌─────────────────────┐                     │
│                 │       로그인        │                     │
│                 └─────────────────────┘                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 기능

- 이메일/비밀번호 입력
- 로그인 버튼 클릭 시 인증
- 성공 시 메인 페이지로 이동
- 실패 시 에러 메시지 표시
